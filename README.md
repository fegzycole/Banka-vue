## About The Project

Banka is a fullstack core banking application. It is comprised of three modules(Client, Cashier and Admin) each with it's own dashboard.


### Built With

- Vue.js

## Prerequisites
 - Git
 - Yarn
 - Node.js
 - NPM

## Getting Started

- git clone https://github.com/fegzycole/Banka-vue.git

- cd Banka-vue

- npm install

- npm run serve
